8.流程控制语句
    顺序
    分支：通过条件判断决定执行的代码

    单分支
        语法：
            if 条件判断(True|False):
                语句
        说明：
            if 是单分支关键字
            根据条件表达式返回的结果决定是否执行语句内容
            如果返回true，语句执行，如果返回False，语句不会被执行
            分支体的内容必须进行缩进
            if后面必须以:结尾

    双分支
        语法：
            if 条件判断:
                语句1
            else：
                语句2
        说明：
            语句1和语句2不可能同时执行
            如果条件判断返回True，执行语句1内容
            如果条件判断返回False，执行语句2内容

    多分支
        语法：
            if 条件判断1：
                语句1
            elif 条件判断2：
                语句2
            elif 条件判断3:
                语句3
                ...
            else:
                以上所有条件判断都没有成立，执行的内容

        说明：
            只会有一个分支被执行
            如果条件判断1成立(返回True)，那么执行语句1内容，其它分支内容不会被执行
            如果条件判断1不成立(返回False),那么继续对条件2进行判断
            如果条件判断2成立(返回True)，那么执行语句2内容
            如果条件判断2不成立(返回False),那么继续对条件3进行判断
            如果条件判断3成立(返回True)，那么执行语句3内容
            如果以上所有条件判断都没有成立，执行else语句的内容

        注意：else最多有一个，elif可以有多个，else语句在所有的else if语句之后
