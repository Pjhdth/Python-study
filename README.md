8.流程控制语句

    顺序
    分支：通过条件判断决定执行的代码

    单分支
        语法：
            if 条件判断(True|False):
                语句
        说明：
            if 是单分支关键字
            根据条件表达式返回的结果决定是否执行语句内容
            如果返回true，语句执行，如果返回False，语句不会被执行
            分支体的内容必须进行缩进
            if后面必须以:结尾

    双分支
        语法：
            if 条件判断:
                语句1
            else：
                语句2
        说明：
            语句1和语句2不可能同时执行
            如果条件判断返回True，执行语句1内容
            如果条件判断返回False，执行语句2内容

    多分支
        语法：
            if 条件判断1：
                语句1
            elif 条件判断2：
                语句2
            elif 条件判断3:
                语句3
                ...
            else:
                以上所有条件判断都没有成立，执行的内容

        说明：
            只会有一个分支被执行
            如果条件判断1成立(返回True)，那么执行语句1内容，其它分支内容不会被执行
            如果条件判断1不成立(返回False),那么继续对条件2进行判断
            如果条件判断2成立(返回True)，那么执行语句2内容
            如果条件判断2不成立(返回False),那么继续对条件3进行判断
            如果条件判断3成立(返回True)，那么执行语句3内容
            如果以上所有条件判断都没有成立，执行else语句的内容

        注意：else最多有一个，elif可以有多个，else语句在所有的else if语句之后

    循环

1.嵌套分支

    在分支中又包含了分支判断

2.match...case分支

    语法:
    match x:
    case a:
    语句1
    case b:
    语句2
    case _:
    语句3

        说明：
            match是关键字
            x是一个表达式，后面的:必须加
            用x表达式返回结果和case后的值进行匹配
            如果和a匹配上了，执行语句1的内容
            如果和b匹配上了，执行语句2的内容
            如果和a、b都没有匹配上，执行语句3的内容

3.三目运算符

    值1 if 条件表达式成立 else 值2

4.循环

    当满足某个条件的时候，重复的执行一段代码
    while
    语法
    while 条件表达式:
    重复执行的代码(循环体)
    else:
    当 while 表达式结果为 False 时会执行 else 中的语句
    说明
    先判断条件是否成立，如果条件成立就执行循环体一次；然后再判断条件是否成立，如果成立，继续执行循环体，直到循环条件不成立的时候，
    才会结束循环，执行循环下面的其他语句。判断条件可以是任何表达式，任何非零、或非空（null）的值均为true。执行语句可以是单个语句或语句块
    如果条件表达式一直成立，那称之为无限循环，也叫死循环
    注意：为了避免死循环，在循环体中别忘了改变影响循环条件的变量

        for
            语法
                for 临时变量  in 可迭代对象:
                    重复执行的代码(循环体)
                else:
                    循环正常结束的时候，执行的代码

            说明
                for、in关键字
                首先判断可迭代对象中是否有下一个元素
                如果有将元素取出，赋值给临时变量
                继续判断是否有下一个元素，直到可迭代对象中没有元素了，循环结束

        如果明确循环的对象或者循环次数使用for循环
        如果不明确循环的对象或者循环次数使用while循环

1.回顾程序流程控制

    顺序
        程序正常执行的顺序
    分支
        当满足某个条件的时候，才会执行代码
        if 条件表达式0:
            语句0
        elif 条件表达式1:
            语句1
        elif 条件表达式2:
            语句2
        else:
            以上所有条件都没有满足的时候，执行的代码

        match 表达式:
            case 值1:
                语句1
            case 值2:
                语句2
            case _:
                以上所有case都没有匹配，执行的代码

        三目运算
            值1 if 条件表达式 else 值2

    循环
        当满足某个条件的时候，重复执行代码
        while
            while 条件表达式:
                循环体
            else：
                循环正常结束，执行的代码；如果通过break跳出循环，else中的内容不会被执行
        for
            for 临时变量 in 可迭代对象:
                循环体
            else:
                 循环正常结束，执行的代码；如果通过break跳出循环，else中的内容不会被执行

        range(start,stop,step):生成一个数列，数量从start开始(包含)，到stop结束(不包含)，以step步长进行生成

    关键字
        continue:跳出当前正在进行的循环，继续下一次循环
        break:跳出整个循环
        pass: 占位语句

    注意：不管是分支还是循环，在分支体或者循环体中定义的变量都可以在外面访问，说明它们都没有增加新的作用范围

3.容器数据类型

    序列
        有序，指的是有先来后到的顺序，放进去的顺序和取出的顺序是一致的
        有序意味着容器中的元素都有下标(索引)
        list、str、tuple
    列表list
        有序
        可变的
        可以重复
        通过[]定义列表
    字符串str
        有序
        不可变的
        可以通过单引号、双引号以及三个引号定义字符串
    元组tuple
        有序
        不可变
        元素可以重复
        通过()定义元组对象

    集合set
        无序，不能通过下标定位元素，不能进行切片操作
        可变
        元素不能重复的
        通过{}定义集合
        注意：如果要定义一个空集合的话，不能使用{},只能使用set()

    字典dict
        以键值对的形式存储数据
        无序，不能通过下标定位元素，不能进行切片操作
        一般我们是通过key获取value
        可变的（key:要求使用不可变类型        value:可以改变）
        通过{}定义字典，字典中的键值对用：进行分隔，多个键值对之间用逗号进行分隔
            {"name":"zs","age":18}

4.函数

    带名字的代码块
    将完成某一个功能的代码封装起来，以后再去使用这个功能的时候，不需要写这些代码，直接调用封装的函数即可
    提供代码的复用性

    函数定义的语法
        def 函数名(参数列表):
            函数完成的功能(函数体)
            [return 返回]

    函数定义的语法说明
        def 是定义函数的关键字
        函数名是程序给当前功能代码取的名字 ，需要遵循标识符的命名规则
        参数列表需要放到小括号中声明，是函数执行的时候需要调用者传递数据,如果没有传递数据，小括号也不能省
        函数声明完毕之后，用冒号结束
        函数体需要进行缩进，里面放的是函数具体实现的功能
        如果函数执行完毕之后，需要给调用者返回内容的话，通过return进行返回

    注意：函数必须要先定义，再使用
        调用函数: 函数名(参数)

5.函数的参数

    形式参数
        函数的提供者在定义函数的时候，声明的参数，叫做形式参数，简称形参
        形参没有分配存储空间，也没有值，相当于一个占位符
    实际参数
        函数的调用者在调用函数的时候，传递的参数，叫做实际参数，简称实参
        传递的是具体的值，底层会执行赋值操作，将实参赋值给形参

6.对于可变数据类型不同赋值方式的区别

    var = var *2  创建新的对象
    var *=2       不会创建新的对象
